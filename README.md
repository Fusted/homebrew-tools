# Homebrew Tap - fusted/tools

Personal Homebrew tap for various tools.

## Installation

```bash
brew tap fusted/tools
```

## Available Formulas

- **flac-to-alac** - Convert FLAC files to ALAC (Apple Lossless) format
  ```bash
  brew install flac-to-alac
  ```

## Repository

Main repository: [fusted/flac-to-alac](https://github.com/fusted/flac-to-alac)
